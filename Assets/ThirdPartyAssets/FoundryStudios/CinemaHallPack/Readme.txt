Thank you for choosing Foundry Studios. Our goal is to support and empower artists and developers by providing high-quality, optimized assets and practical solutions for real-time projects. This package contains a collection of modular, realistic environment assets designed for use in real-time Unity projects. The assets are suitable for games, simulations, and cinematic scenes.

FEATURES

- Modular environment and prop assets
- Real-world consistent scale
- Optimized for real-time performance
- Prefabs ready for drag and drop use

COMPATIBILITY

- Supported minimum Unity Version: 2021.3 LTS
- Supported Render Pipelines:
  - Built in Render Pipeline
  - Universal Render Pipeline (URP)
  - High Definition Render Pipeline (HDRP)
- All default Unity lit shaders, no custom shaders or maps

TIPS

- Models can be subdivided for more detail
- Texture resolution can be turned down for better performance
- Most of the colliders are optimized (box or convex colliders) but they can be replaced with mesh colliders for more detail

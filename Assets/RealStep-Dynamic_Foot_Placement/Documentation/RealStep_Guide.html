Quick Start

1
Ensure your character has an Animator with a Humanoid avatar and IK Pass enabled on the Animator Controller layer.
2
Add Foot IK via Add Component → RealStep → Foot IK.
3
Set the Ground Layer to match your terrain/ground layers.
4
Press Play — feet will automatically adapt to slopes, stairs, and uneven surfaces.
Tip: Use the Help button in the inspector header to open this documentation. Use Show Gizmos to visualize raycasts in Scene view.

Features

  • Foot IK Placement — Feet adapt to ground height and rotation on slopes, stairs, and uneven terrain.
  • Pelvis Adjustment — Body height adjusts automatically to match foot positions.
  • Body Tilt — Character body tilts to match ground slope with smooth interpolation.
  • Per-Foot Weight — Independent IK weight control for left and right foot.
  • Auto Max Step — Automatic step height detection based on character size.
  • Smooth Enable/Disable — Smoothly blend IK on/off via DisableSmooth().
  • Foot Events — UnityEvents for foot grounding and lifting (footsteps, particles, etc).
  • Edge Detection — Normal validation and hysteresis prevents glitching at edges.
  • Debug Gizmos — Visualize raycasts and live grounded state in the inspector.

Events

RealStep fires UnityEvents when foot grounding state changes. Use these to trigger footstep audio, particles, or gameplay logic.

EventFires When
OnLeftFootGroundedLeft foot touches ground
OnLeftFootLiftedLeft foot leaves ground
OnRightFootGroundedRight foot touches ground
OnRightFootLiftedRight foot leaves ground
